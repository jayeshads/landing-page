# Meta Campaign Best Practices & Guidelines

## Campaign Structure & Objectives
1. Use `OUTCOME_LEADS` objective for lead generation campaigns (forms, landing pages, WhatsApp).
2. Use `OUTCOME_SALES` objective for direct e-commerce purchase conversions.
3. Recommend CBO (Advantage+ Budget Budget Optimization) when testing multiple ad sets.
4. Set daily budget minimums based on target CPL (at least 3-5x estimated CPL per day).

## Placement Best Practices
- Combine Facebook Feed, Instagram Feed, Instagram Stories, and Reels for maximum reach.
- For video creatives, prioritize 9:16 vertical aspect ratio for Instagram Stories & Reels.

## Optimization Rules
- Initial learning phase requires ~50 conversions per ad set per week.
- Keep audience sizes between 1M to 5M for location-targeted campaigns in India.
